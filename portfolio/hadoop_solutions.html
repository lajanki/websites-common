Hadoop

The following is a collection of solutions to selected excercises of Udacity's Intro to Hadoop and MapReduce course.

Store purchases (Lesson 3 Project Part I)

Given a list of purchases made via a credit card accross different stores, write MapReduce programs to answer the following problems. Input data: purchases.txt.gz (links to content.udacity-data.com, 38.5MB)

By category Provide a sales breakdown by product category across all of our stores. mapper.py reducer.py
Highest sales Find the monetary value for the highest individual sale for each separate store. mapper.py reducer.py
Total sales Find the total sales value across all the stores, and the total number of sales. Assume there is only one reducer. mapper.py reducer.py

Web log (Lesson 3 Project Part II)

Given a typical web server log file, write MapReduce programs to answer the following problems. Input data: access_log.gz (links to content.udacity-data.com, 54.5MB)

Page hits Write a MapReduce program which will display the number of hits for each different file on the Website. mapper.py reducer.py
Hits from Write a MapReduce program which determines the number of hits to the site made by each different IP address. mapper.py reducer.py
Most popular file Find the most popular file on the website, ie. the file listed the most number of times in access_log. mapper.py reducer.py

User forum data (Final project)

Given a dump of the Udacity user forums as .tsv files, write MapReduce programs to answer the following problems. Input data: forum_data.tar.gz (links to content.udacity-data.com, 39.7MB)

Peak hour Find for each student what is the hour during which the student has posted the most posts. mapper.py reducer.py
Post lengths Write a MapReduce program that would process the forum_node data and output the length of the post and the average answer length for each post. mapper.py reducer.py
Top tags Write a MapReduce program that would output the Top 10 tags, ordered by the number of questions they appear in. mapper.py reducer.py
Posters Write a MapReduce program that for each forum thread gives a list of students that have posted there. mapper.py reducer.py
Combining posters with posts Write mapper and reducer code that combines forum post data with reputation data of the poster. (Lesson 4 Quiz: Combine Datasets) mapper.py reducer.py
