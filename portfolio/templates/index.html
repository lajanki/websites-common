Game Descriptor

							Website that writes game descriptions based on Steam store


						
Details

							A Markov chains based text generator trained against a sample
							of Steam Store game descriptions. Includes randomly generated game tags
							and a pattern based developer name.

							The website is a Flask application running on Google App Engine. The generator is re-trained weekly using a random
							set of game descriptions.

Tweet Mapper

							Website for mapping tweets to a Google Map and showing trending topics

						
    
							
  • Find tweets near map coordinates
    • 
							
  • Plot recent timeline on a map
    • 
							
  • Show recent timeline statistics
    • 
							
  • Show worlwide and country specific trending topics
    • 
						

						
Details

							A custom map using Google Maps JavaScript API connected to a
							PHP server script querying Twitter's Search API for tweets. Clicking on the
							map will attempt to find a random sample of tweets near that location. Can also be used to map a user's recent timeline and computes
							some user statistics from that timeline.
						

						

							Trending data is pulled from Twitter periodically and stored in a local cache on the server. Additionally, a database is used to
							store recent history of a trending topic and displayed in a chart.

Date Profiler

							Website generating online dating profiles by filling templates using Google autocomplete

						

							
I love to socialize, embrace life and I am using a loat of electricity if I say so myself!

						


						
Details

							The backend is a Flask application filling a set of dating profile samples labelled with prefixes acting as input
							to Google's autocomplete API. Profile templates are filled from a local cache of autocomplete responses in order
							to reduce latency and outgoing API calls. The cache is refreshed weekly. Powered by Google App Engine.

@little_youtube

							Twitter bot searching for old YouTube videos with no views



						
Details

							A Python script performing a YouTube search using the YouTube Data API.
							Since view count cannot be used as a search parameter, the scripts reads a batch of search terms from a dictionary file,
							performs the search and stores links to zero view items to a result file. The links are then tweeted one by one until

About

				
				
I am a mathematician interested in data science and programming. I received my master's degree in mathematics
					in 2013 with a minor in computer science.
				


				
In spring 2015 I bought a Raspberry Pi Model 2 in order to teach myself some PHP and web server basics.
					Rather than doing that I quickly found myself toying with the camera module creating timelapse videos and reading about
					the things other people used their Pis.
				


				
Around that time I started paying attention to content with clever ideas behind them posted to Reddit.
					PetitTube, a website that streams YouTube videos no one has ever seen, was an early
					inspiration to start figuring out how to build such a system and to put the Pi to actual use. I had already grown to like
					Python but had no experience making HTTP requests or using website APIs. After figuring out how to read the (rather extensive)
					documentation Google has for their YouTube Data API, I managed to make something similar. I chose to use Twitter as a platform
					since writing a Twitter bot seemed to be a popular way to get started with a Pi.
				


				
Eventually, I did learn how Apache works and how to make AJAX requests to a server script when I needed a website
					as a frontend to one of my projects. I was not surprised to learn that neither PHP nor JavaScript are among people's favorite languages.
				


				
I've also written some articles for my math page. They are mostly
					to teach myself new concepts but occasionally to give a deeper background to a high school level concept.
				


				
Currently I'm looking to shorten the list of math articles to write about as well as transforming one of my
					Pi bots to better utilize data analysis tools to solve a certain problem.
